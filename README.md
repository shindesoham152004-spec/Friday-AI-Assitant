# Friday-AI-Assitant
Friday is a Python-based AI-powered voice assistant that enables users to interact with their system using natural speech. It can perform tasks such as opening websites, playing music, searching the web, fetching real-time weather information, and answering general queries using AI-generated responses.

The assistant integrates speech recognition, text-to-speech, and Google Gemini AI to provide intelligent and concise replies. With support for multiple APIs and hands-free interaction, Friday offers a smooth and interactive voice-controlled assistant experience.
